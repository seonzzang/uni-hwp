Features
왜 uni-hwp인가요?
⚡

Rust 기반 고성능

Rust로 구현된 HWP 파서 엔진이 WebAssembly로 컴파일되어, 브라우저에서도 네이티브 수준의 빠른 렌더링을 제공합니다.

🌐

완벽한 크로스 플랫폼

웹 브라우저, Windows 데스크톱(Tauri), macOS, Linux 등 어떤 환경에서도 동일한 편집 경험을 제공합니다.

📄

HWP/HWPX 완벽 지원

한글과컴퓨터의 HWP 및 HWPX 형식을 완벽하게 지원하며, 문서의 서식과 레이아웃을 정교하게 재현합니다.

🔒

프라이버시 보호

모든 문서 처리가 로컬 환경에서 이루어집니다. 서버로 문서가 전송되지 않아 보안에 민감한 문서도 안심하고 사용할 수 있습니다.

✏️

강력한 편집 기능

텍스트 편집, 표 조작, 이미지 삽입, 머리말/꼬리말, 각주, 스타일 등 전문적인 문서 편집 기능을 지원합니다.

💻

오픈소스

모든 코드가 GitHub에 공개되어 있습니다. 누구나 기여하고, 수정하고, 자유롭게 사용할 수 있는 진정한 오픈소스 프로젝트입니다.
